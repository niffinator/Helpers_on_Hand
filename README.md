Helpers on Hand Installation and Setup

Establish Database
1. Right click to open XAMPP Control Panel Run as Administrator
2. Start Apache
3. Start mySQL
4. Minimize Contol Panel
5. Open internet browser
6. Type in URL:  localhost
7. Hit enter
8. Click phpMyAdmin from navigation menu
9. Click SQL from top menu
10. Open GitHub repository Helpers_on_Hand
11. Open file: HOH_SQL.txt
11. Copy entire file content
12. Paste into SQL window in phpMyAdmin SQL window
13. Click Go
  
Open Interface
1. New web browser window
2. URL: localhost/HOH_Home_Page.php
